## Getting Started

1. Clone the repository:
    ```bash
    git clone https://github.com/mohneesh-gupta/CivicLens.git
    ```
2. Install dependencies:
    ```bash
    npm install
    ```
3. Start the application:
    ```bash
    npm start
    ```
